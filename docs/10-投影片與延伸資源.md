## 如何使用這些範例

建議不要一開始就複製整套網站或自動化流程。先挑一個小任務：

1. 把一份講義放到 NotebookLM。
2. 把一份 README 放到 GitHub。
3. 用 AI 產生一份教案草稿。
4. 讓 AI 幫你檢查一份教材的可能錯誤。

小任務跑順後，再考慮 agent CLI、Skill、MCP、自動上線等進階工具。
