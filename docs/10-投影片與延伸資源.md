## 如何使用這些範例

建議不要一開始就複製整套網站或自動化流程。先挑一個小任務：

1. 把一份講義放到 NotebookLM。
2. 在 Git 中存一版教材副本。
3. 在工作區指令檔寫好規矩，讓 agent 只讀不改。
4. 讓 AI 幫你檢查一份教材的可能錯誤。
5. 確認後再讓 agent 修改，並用 Git 檢查差異。

小任務跑順後，再考慮 Skill、MCP、自動上線等進階工具；若中途反覆摩擦，先請 agent 整理 Skill 草稿。
