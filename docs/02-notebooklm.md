# 02 NotebookLM 教學

[NotebookLM](https://notebooklm.google.com/) 是 Google 的「根據你提供的來源回答」工具。它很適合投影片中說的「給目錄，不要把全部硬塞進去」：你把講義、PDF、網址放進 notebook，之後問答就有範圍與來源。

## 適合做什麼？

- 幫老師快速查「某個概念在哪份講義」。
- 幫學生根據課程材料複習。
- 從講義產生學習指南、FAQ、練習題、字卡。
- 用心智圖看整章概念的結構。
- 產生語音導讀，讓學生通勤或複習時聽。

## 最小上手流程

1. 打開 <https://notebooklm.google.com/>。
2. 建立新 notebook，命名為課程或章節名稱。
3. 加入來源：PDF、Google Docs、Slides、網站連結或文字。
4. 先問：

```text
請列出這些來源共同涵蓋的核心概念。
每一點都要附來源引用。
```

5. 再問：

```text
請根據來源產生一份學生用學習指南。
包含：重點整理、常見誤解、5 題練習題。
不要使用來源以外的內容。
```

## 學生端分享方式

分享前先檢查：

- 來源是否有權限分享給學生。
- 是否包含學生個資、成績、作業批改紀錄。
- 是否包含未授權公開散布的書籍或講義。

建議分享文字：

```text
這是本章的 NotebookLM 複習筆記本。
你可以用它查講義內容、產生學習指南、做練習題。
請注意：它只是一個輔助工具，正式答案仍以課堂與公告為準。
```

## 推薦問題

```text
這份講義最重要的 5 個概念是什麼？
```

```text
如果學生第一次學這個主題，最容易誤解哪三點？
```

```text
請根據來源產生 10 題選擇題，並附上答案與來源引用。
```

```text
請把這章整理成一張概念地圖的文字大綱。
```

```text
請列出我上傳來源之間的矛盾、缺漏或需要人工確認的地方。
```

## 跟 agent 的分工

| 任務 | NotebookLM | 桌面版 agent / CLI |
|---|---|---|
| 根據講義問答 | 很適合 | 可以，但需要先讀檔 |
| 產生學習指南 | 很適合 | 可以 |
| 修改本機檔案 | 不適合 | 很適合 |
| 產生多版本考卷 | 不適合 | 很適合 |
| 部署課程網站 | 不適合 | 很適合 |

## 常見限制

- NotebookLM 擅長「讀與答」，不是自動幫你改檔案。
- 來源錯，它的回答也會跟著錯。
- 分享 notebook 前要確認權限與個資。
- 它可能摘要得很漂亮，但仍需要教師最後判斷。
