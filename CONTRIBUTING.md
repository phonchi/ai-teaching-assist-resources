# 貢獻方式

歡迎補充工具、修正連結、加入更清楚的新手教學。

## 可以貢獻什麼？

- 新增投影片相關工具的官方入口。
- 補充 Windows/macOS/Linux 安裝注意事項。
- 加入教師可直接複製的 prompt 範本。
- 補充 Skill、MCP、subagent 的教學案例。
- 修正過期連結或錯誤說法。

## 撰寫風格

- 使用繁體中文。
- 優先給「第一步怎麼做」，不要只講概念。
- 外部工具請使用官方文件或主要 GitHub repo。
- 若是非官方工具，請明確標示「非官方」與風險。
- 不放學生個資、成績、未授權教材。

## 新增工具格式

```markdown
## 工具名稱

- 入口：<https://example.com/>
- 用途：
- 適合誰：
- 最小上手：
- 注意事項：
```

## 發 PR 前檢查

- [ ] 連結可打開。
- [ ] 內容與「AI 教學助攻」主題相關。
- [ ] 沒有放入私密資料。
- [ ] 新增文件已從 README 或其他 docs 連到。
